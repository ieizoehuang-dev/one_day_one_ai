# 一天一AI — 文章摘要索引

數位時代「[一天一AI](https://www.bnext.com.tw/tags/%E4%B8%80%E5%A4%A9%E4%B8%80AI)」專欄的文章索引，以互動式 HTML 捲動式單頁文件呈現。

---

## 專案內容

本專案自動追蹤數位時代「一天一AI」專欄，將文章整理成兩個同步格式：

| 檔案 | 用途 |
|------|------|
| `onedayoneai.md` | 純文字索引，含主題分類與文章表格 |
| `onedayoneai.html` | 互動式捲動頁面，支援深色／淺色模式切換 |

---

## 互動式頁面功能

開啟 `onedayoneai.html` 即可使用：

- **捲動瀏覽**：由上至下依序為更新摘要、主題分類、第一至三頁文章表格
- **深色 / 淺色模式**：右上角切換按鈕
- **新文章標示**：最新一批新增文章以銅橘色左邊框標示
- **分類連結**：主題分類區塊可直接點擊跳至原文

---

---

## 更新方式

本專案搭配 **Claude Code** 自動化更新。在 Claude Code 中輸入「請更新」，即會自動：

1. 抓取數位時代網站最新文章清單
2. 比對現有索引，識別新增文章
3. 同步更新 MD 與 HTML 兩個檔案
